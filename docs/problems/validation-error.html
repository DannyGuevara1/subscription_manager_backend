Validation Error (400 Bad Request)

Este error ocurre cuando los datos enviados en la petición no cumplen con las reglas de validación del servidor.

¿Qué hacer?

  • Revisa el campo 'extensions' en la respuesta del error para ver los detalles específicos.
  • Asegúrate de que todos los campos requeridos estén presentes.
  • Verifica que el formato de los datos sea el correcto (ej. un email válido).
